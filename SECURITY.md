
# Security Policy

## Supported Versions
Security fixes will be applied to the `main` branch.

---

## Scope
Because TrialForge AI is a research/simulation tool, security considerations focus on:

- API key handling  
- Preventing prompt injection attacks in the protocol parser  
- Avoiding execution of user-supplied code  
- Protecting simulation endpoints from misuse  
- Ensuring environment variables are never logged  

---

## Responsible Disclosure
We follow standard responsible disclosure practices.  
Researchers who responsibly report vulnerabilities will be acknowledged.
